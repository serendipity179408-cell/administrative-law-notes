徐偉超・行政法・第二堂課

私經濟行政區別實益與公私法區分理論

課程範圍概覽

接續上一堂課的「私經濟行政」議題，本堂課以釋字第457號為核心案例，深論行政私法受憲法基本權拘束的論證脈絡，再依序展開：

  1. 公權力行政與私經濟行政的八項區別實益（基本原則、組織形態、行為方式、程序規範、強制執行、救濟途徑、損害賠償）
  2. 干預行政與給付行政的定義、區別實益，以及給付行政的組織／行為方式選擇自由
  3. 公法與私法區分的五個主要學說（利益說、隸屬說、主體說、新主體說、實質主體說）與兩個輔助標準（傳統說、事理關聯說）
  4. 事件性質判斷的操作流程
  5. 行政法的性質（公法、國內法）與類型（普通/特別、實體/程序、內部/外部）
重點提示

一、釋字第457號深論：退輔會配耕案

案件事實

政府遷台初期，為安置除役榮民生活，退輔會將所經營之國有農場耕地配予榮民耕種。本案發生於屏東農場：榮民（場員）收養一養女，養女長大後出嫁；場員死亡後，養女向退輔會申請繼續耕作已故養父所配耕之土地，卻遭退輔會依「本會各農場有眷場員就醫、就養或死亡開缺後房舍土地處理要點」第四點第三項駁回——該要點規定：兒子不論結婚與否均得繼承配耕權利，女兒則限於未出嫁方得繼承，已出嫁者無條件收回土地房舍。當事人窮盡訴願、行政訴訟、民事訴訟三審程序均遭駁回後，聲請釋憲。

憲法層次論證

該要點僅以性別與已否結婚，對女兒為差別待遇，非基於男女生理差異之合理差別待遇，違反憲法第7條暨增修條文第10條第6項之男女平等原則、兩性地位實質平等原則。

用語提醒
解釋文結論用語為「與前述原則不符」，並命主管機關「於解釋公布之日起六個月內，基於上開解釋意旨，就相關規定檢討，妥為處理」，並未使用「違憲」二字，也未宣告該要點自公布日起失其效力。效果上接近違憲宣告，但用語與釋443、釋455等直接宣告「失其效力」的案型不同，作答時建議寫「認定與男女平等原則有違，命檢討處理」較為精確。
行政法層次論證：行政私法受基本權拘束

第一步：配耕本身屬於行政私法——國家基於照顧榮民生存照顧義務，以私法形式提供給付，退輔會與配耕榮民間成立使用借貸之法律關係（民法第464條，無償契約）。

第二步：配耕榮民死亡後，其繼承人可否繼續耕作，性質上等同國家對繼承人另為一次給付，退輔會就此應「依男女平等原則妥為規劃」——這句話正是本案的關鍵：它意味著私經濟行政（行政私法）仍應受憲法基本權（此處為平等權）之拘束，國家不能藉由變換為私法契約形式（使用借貸）而規避憲法拘束。

釋字457號解釋文重點摘錄
「國家機關為達成公行政任務，以私法形式所為之行為，亦應遵循上開憲法之規定」；「受配耕榮民與國家之間，係成立使用借貸之法律關係」；「上開房舍土地處理要點第四點第三項……其中規定限於榮民之子，不論結婚與否，均承認其所謂繼承之權利，與前述原則不符」。
一句話記憶
國家換了衣服（私法形式），骨子裡的憲法義務（基本權拘束）沒有換——這就是本案在「私經濟行政是否受基本權拘束」爭議中，通說「行政私法理論」的實務依據。

二、公權力行政與私經濟行政之區別實益

八項區別實益總表

依「基礎理論／行政組織／行政作用／行政救濟」四大區塊分類如下：

類屬區別項目公權力行政（以徵兵、課稅為例）私經濟行政（以行政輔助為例）
基礎理論基本原則依法行政原則拘束私法自治、契約自由
行政組織組織形態公法組織（公法人＋行政機關）公法組織或私法組織（如中油、台電等股份有限公司）
行政作用行為方式行政處分為大宗，另有行政命令、單純高權行政契約、事實行為僅能締結私法契約，不能做成行政處分
程序規範行政程序法（§2 Ⅰ）無行政程序法適用，行政輔助取得物品能力適用政府採購法
強制履行依據行政執行法強制執行法
行政救濟救濟途徑行政訴訟法／行政法院民事訴訟法／普通法院
損害賠償國家賠償法民法侵權行為或債務不履行規定
易混淆考點
公私經濟行政「均適用行政程序法」是錯誤敘述——只有公權力行政才適用行政程序法（行政程序法第2條第1項僅規範行政處分、行政契約、法規命令與行政規則、行政計畫、行政指導、陳情等行為之程序）。
歷屆考題演練
題目一：公權力／私經濟行政區別實益之敘述何者錯誤？

(A) 私經濟行政爭議應依民事訴訟法 (B) 公權力行政所受損害可依國家賠償法請求賠償 (C) 私經濟行政受損害依民法請求賠償 (D) 公私經濟行政均同樣適用行政程序法

答案：(D)
(A)(B)(C) 均正確；(D) 錯誤，僅公權力行政適用行政程序法。
題目二：何種行政任務不得以私法形式為之？

(A) 提供青年創業無息貸款 (B) 提供民生自來水 (C) 提供低收入戶社會住宅 (D) 設置派出所維護地方治安

答案：(D)
(A)(B)(C) 均屬給付行政，可用公法或私法形式；(D) 屬干預行政，不得以私法形式為之。

三、干預行政與給付行政

定義與實例

依行政措施手段所生效力區分：

  • 干預行政：課加人民義務及負擔、限制自由及權利。例：徵兵（限制人身自由）、課稅（限制財產權）。
  • 給付行政：提供給付或授予利益。例：核發建照、核發專利、核發畢業證書。

部分行政任務兼具兩者，例如國民教育：強迫入學部分屬干預，教授知識內容部分屬給付。

區別實益
區別項目干預行政給付行政
法律保留原則一律適用（須有法律依據，如兵役法、所得稅法）僅「重大事項」適用（釋字第443號解釋理由書）
組織與行為方式僅能用公法組織＋公法行為，形成公法關係具「組織與行為方式選擇自由」，可自由組合
釋字443號解釋理由書重點（已查證）
「又關於給付行政措施，其受法律規範之密度，自較限制人民權益者寬鬆，倘涉及公共利益之重大事項者，應有法律或法律授權之命令為依據之必要，乃屬當然。」——此即「層級化法律保留」理論在給付行政上的具體展現：非重大事項的給付行政無須法律保留。
給付行政的組織與行為方式選擇自由（三圈圖） 
給付行政的三種組織／行為方式組合：

【圈圈一】公法組織 ＋ 公法行為 → 形成公法關係
          例：社會局以行政處分核發低收入戶生活補助
              健保署與特約醫事機構締結行政契約

【圈圈二】公法組織 ＋ 私法行為 → 形成私法關係
          例：釋字457號（配耕）、釋字540號（國宅租售）
              台北市自來水處供水（公法組織＋私法供水契約）

【圈圈三】私法組織 ＋ 私法行為 → 形成私法關係
          例：台灣電力公司供電契約
              台灣省自來水公司供水契約

圈圈一形成公法關係，圈圈二、三形成私法關係——這就是「給付行政具有法律形式選擇自由」的具體操作。

易混淆考點：自來水的公私法定性
同樣是提供自來水，台灣省自來水公司用私法組織（圈圈三），台北市自來水處則用公法組織搭配私法行為（圈圈二）——理論上圈圈一二三都可以，但實務上因地制宜，不可一概而論。

四、公法私法區分學說（專題一）

五個主要標準

1. 利益說

依法規所保障的利益是公益或私益判斷。批評：①公共利益概念本身不明確 ②現代型公法往往兼具保障公益與私益，無法發揮判斷功能。

2. 隸屬說（從屬說，亦稱權力說）

依法律關係是上下隸屬（公法）或對等關係（私法）判斷。批評：公法除了上下也有對等（如行政契約）；私法除了對等也有上下（如民法親屬編父母懲戒子女）。此說僅證實：傳統典型公法規範確為上下隸屬關係。

3. 主體說（舊主體說）

規範主體一方為國家即為公法；雙方均為私人即為私法。致命批評：民法規範私人也規範國家（如國家從事私經濟行政適用民法），依此說民法會被推論為「公法」，形成私法質變為公法的謬論，無人能接受。

4. 新主體說（形式主體說，目前通說）

依權利義務的歸屬對象判斷：僅以國家為歸屬對象者為公法；非僅以國家（即國家或人民皆可）為歸屬對象者為私法。此說修正主體說「僅」一字，解決民法定性問題。批評：①遇到行政組織法（規定的是組織而非權利義務）無法發揮判斷功能 ②現代型公法（如所得稅法）同時規範國家與人民，依此說反而會被推論為私法。

5. 實質主體說（補充見解，非所有教材共同列入）

法規範所規範的主體，至少一方為高權主體，並就其高權主體地位規定權利義務或組織者，即為公法。此說同時解決新主體說的兩個缺點：只要至少一方為高權主體即可（處理現代型公法同時規範國家人民的問題），且將「組織」也納入規定範圍（處理行政組織法無法判斷的問題）。

學說發展脈絡
每一說都在救援前一說的缺點，但最後都有自己的缺點。目前通說教科書多數採新主體說；實質主體說屬部分學者（曾華松大法官、相關期刊論文）補充見解，考試作答仍以新主體說為主軸即可。
兩個輔助標準

當五個主要標準無法發揮判斷功能，或各標準判斷結果不一致時，動用輔助標準：

1. 傳統說（一依往昔）

依歷來慣例認定其公私法屬性。

2. 事理關聯說（事件關聯說）

若某屬性不明的B法規，與另一屬性明確的A法規具有事理上的密切關聯（如特別法／普通法、前法／後法、母法／子法關係），則B法規從A法規之屬性認定。

公法私法區別實益（近似於公權力／私經濟行政區別實益）

公法私法的區別實益「近似於」（非完全等於）公權力／私經濟行政的區別實益——前者針對「法規屬性」，後者針對「行政類型」，兩者範疇不同，故不能完全劃等號。

獨有的區別實益：事務權限之歸屬（憲法第107條第3款）
公法事務之規範制定權限歸屬於中央（中央可定法律、法規命令，地方亦可訂自治條例、自治規則）；私法事務之規範制定權限僅限中央（地方不可規範）。依憲法第107條第3款：「國籍法及刑事、民事、商事之法律」由中央立法並執行——民商法即私法，其立法權專屬中央。

其餘區別實益（行為方式、程序規範、強制執行、救濟途徑、損害賠償）則與公權力／私經濟行政區別實益相同，此即「近似於」的由來。

五、事件性質之判斷（專題二）

判斷流程：如何認定社會生活紛爭事件屬公法事件或私法事件？

判斷實益：僅一項——救濟途徑（公法事件走行政法院，私法事件走普通法院）。

事件有法規可適用？
├─ 是 → 依法規屬性決定事件性質
│        （法規屬性判斷回到「專題一」：五主要標準＋兩輔助標準）
│
└─ 否，或有多數屬性不同法規可適用 → 另闢蹊徑：
     │
     ├─ 屬秩序行政／課稅行政（干預行政性質）→ 逕認定為公法事件
     │   （上對下、具高權色彩）
     │
     └─ 屬給付行政 →
          ① 先依行政機關的意思（機關認定公法即公法，私法即私法；
             理由：給付行政本有法律形式選擇自由）
          ② 機關意思不明 → 依事件的脈絡客觀認定
             （公法脈絡特徵：具公益色彩、具高權／單方規制性）

六、行政法之性質與類型

行政法的性質

行政法的「性質」（本質）不同於「特徵」——特徵是與他者比較而來，性質是就自身觀察其「是什麼」。行政法性質有二：

  1. 公法：規範國家與人民間之權利義務。
  2. 國內法：適用範圍限於一國之內。

行政法作為「規範行政的所有成文與不成文法規範的總稱」，其判斷個別法規是否為公法，即回到「專題一」五主要標準＋兩輔助標準。

行政法的類型：三組區分
區分區別標準類型與例示區別實益
普通行政法／特別行政法適用範圍普通：適用於所有行政領域；特別：僅適用部分行政程序（如警察職權行使法）特別法優於普通法，優先適用特別行政法
行政實體法／行政程序法規範內容實體：規範國家人民間權利義務（兵役法、所得稅法）；程序：實現權利義務之方法（行政訴訟法）實體從舊、程序從新
內部行政法／外部行政法適用對象外部：以人民為對象；內部：以行政主體、機關及人員為對象法源不同：外部法源為憲法、法律、法規命令；內部法源另加行政規則

案件事實

藍女士（本姓王）於民國68年結婚後，於77年被國軍退除役官兵輔導委員會屏東農場場員藍先生收養為養女，並與養父共同耕作其配耕之國有農場土地。養父於79年病逝後，藍女士向屏東農場申請繼耕，遭以「本會各農場有眷場員就醫、就養或死亡開缺後房舍土地處理要點」第四點第三項「死亡場員之遺眷如改嫁他人而無子女者或僅有女兒，其女兒出嫁後均應無條件收回土地及眷舍」為由拒絕；兒子則不論結婚與否均得繼承。藍女士窮盡訴願、行政訴訟（遭認定為私權爭執應循民事程序）、民事訴訟三審後聲請釋憲。

解釋意旨

大法官指出：國家機關以私法形式達成公行政任務之行為，仍應遵守憲法男女平等規定；退輔會與配耕榮民間成立使用借貸之法律關係（私法關係）；配耕榮民死亡後，主管機關對其遺眷是否延續給付，應依男女平等原則妥為規劃；系爭要點僅以性別及已否結婚差別待遇女兒，與男女平等原則有違，主管機關應於六個月內基於解釋意旨檢討，妥為處理。

學理意義

本號解釋是「行政私法理論」在實務上最重要的依據：國家縱以私法外衣（使用借貸）履行具公益、生存照顧色彩的任務，仍不能規避憲法基本權拘束，即「避免公法遁入私法」的具體實踐。

一句話記憶
國家換了衣服（私法），但骨子裡的義務（憲法拘束）沒有換。
釋字第443號｜役男出境限制案（層級化法律保留）

案件事實

內政部依行政院徵兵規則授權訂定「役男出境處理辦法」，於第8條限制役男出境，欠缺法律或法律明確授權之依據，聲請人循訴訟途徑無果後聲請釋憲。

解釋意旨

大法官建立「層級化法律保留」體系：①憲法保留事項（如人身自由核心內容）連立法機關亦不得以法律限制；②國會保留（絕對法律保留）——剝奪生命或限制人身自由須遵守罪刑法定主義；③相對法律保留——限制其他自由權利應由法律規定，或法律具體明確授權命令補充；④給付行政之重大事項，亦須法律或法律授權命令為依據；⑤細節性、技術性次要事項則毋須法律保留，得由命令規範。役男出境限制屬對遷徙自由之重大限制，欠缺法律授權依據，故違憲，應於六個月內失其效力。

學理意義

本號解釋確立法律保留原則並非「全有或全無」，而是依規範對象、內容、限制輕重程度形成層級化體系，給付行政的法律保留密度即由此而來（僅重大事項需要法律保留）。

憲法第107條｜中央立法並執行事項

憲法第107條列舉13款專屬中央立法並執行之事項，其中第3款「國籍法及刑事、民事、商事之法律」，因民商法屬私法性質，故私法事務之規範制定權限僅歸屬中央，地方不得就此另訂自治法規；此與公法事務（地方亦得訂定自治條例、自治規則）形成對照，構成公法私法區別實益中「事務權限歸屬」之規範依據。

行政程序法第2條、第137條

第2條第1項：「本法所稱行政程序，係指行政機關作成行政處分、締結行政契約、訂定法規命令與行政規則、確定行政計畫、實施行政指導及處理陳情等行為之程序。」——此條文明定行政程序法僅適用於前揭六類公權力行政脈絡下的行為，故私經濟行政原則上無行政程序法之適用。

第137條第2項：規範「代替行政處分之行政契約」——行政處分之作成，行政機關無裁量權時，代替該行政處分之行政契約所約定之人民給付，以得為附款者為限。此為「羈束行政不得締結行政契約代替行政處分，裁量行政則可」之規範依據。

兩段逐字稿對第二個學說的稱呼不一致（權利說／權力說），法學通說此說正確名稱為隸屬說（從屬說），「權力說」為常見別名，惟「權利說」並非通用稱呼，本講義已統一採「隸屬說」。
用語提醒：釋字457號並未使用「違憲」字眼
老師課堂簡化說法為「宣告違憲」，惟解釋文實際結論為「與男女平等原則有違」並命主管機關檢討處理，並未逕以「違憲」定性、亦未宣告失其效力。作答時建議精確引用解釋文用語。
